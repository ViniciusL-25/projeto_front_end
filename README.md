Esse projeto consiste em cadastro de alunos por meio do uso de validação e uso de uma API

Utiliza-se de React, JavaScript, JSON, Context API, Docker e Vite.
Em sua programação tem um sistema de navegação de Rotas utilizando o Route e o Navigate. 
Tem uma proteção de rotas também para a página home.
E um localStorage para caso a pessoa ainda permanessa com o Token ativo enquanto estiver com o Login ativo. 
Sua estilização é feita com CSS.

// Como utilizar 
Para conseguir acessar você deve abrir dois terminais

No primeiro terminal utilizar o comando npm install antes dos demais

1. Primeiro terminal entrar na pasta 'cadastro-alunos' e utilizar os comandos utilizar o comando 'npm install' antes dos demais, após ele utiliazar esses 'docker-compose up -d' e depois 'npm run dev'.
Para acessar o vite

2. Segundo terminal acessar a pasta 'cadastro-alunos' depois na pasta 'api' e utilizar o seguinte comando 'npx json-server db.json'.
Para ligar a API

Após fazer todos esses comandos você consiguirá ter acesso total e todas as funcionalidades funcionando.
